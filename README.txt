Secret Santa organizer made for Kizuna.

Usage:
1. Within file, specify location of the .xlsx spreadsheet containing information from the survey. (line 16)
2. In line 17, specify the desired location of the output file which describes the assignment chain.
3. Make sure that the columns correspond to the interpretation as defined in the Participant class.
4. Make sure that the following information is accurate:
    * Current year in line 81
    * Winter formal date and location in line 83
    * Name of previous output file to be removed (in the event that a test was run)
